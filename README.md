## Using your project images (logo & icon)

Place the provided PNG images into the project's `public/` folder to use them as the site icon and full logo. The app already contains SVG fallbacks so the site will show an icon/logo immediately, but you can add the exact PNGs you attached.

- Icon (browser tab / app icon): save the attached PNG as `public/project-icon.png`
- Full project logo: save the attached PNG as `public/project-logo.png`

After placing the files, restart the dev server or refresh the page. You can verify the files are served by visiting:

\`\`\`
http://localhost:3000/project-icon.png
http://localhost:3000/project-logo.png
\`\`\`

If you prefer to keep SVG fallbacks, no action is required — the project already contains `public/project-icon.svg` and `public/project-logo.svg` which will be used automatically.
